Data-TestID Viewer

Es un complemento para la búsqueda del atributo data-testID que se usar en las pruebas.

Este proyecto es un fork de CSSViewer:
source code is publicly available on GitHub for anyone to help and contribute.
https://github.com/miled/cssviewer.


Version 1.0:

How to use :

  • Activar el icono de la extension, mueve el mouse sobre los elementos y mostrará el value de data-testID

  • For quick usage you may use the available keyboard shortcuts:

      [f] Freeze/Unfreeze the widget in place,
      [esc] Disable the viewer.




How to uninstall :

  1. Click the wrench icon on the browser toolbar.
  2. Click Tools.
  3. Select Extensions.
  4. Find the CSSViewer extension then click Uninstall.
